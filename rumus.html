Rumus Excel Dasar

Pelajari rumus-rumus dasar Excel untuk meningkatkan produktivitas kerja Anda

Rumus IF

IF Dasar

  • Sintaks
    =IF(logical_test, value_if_true, value_if_false)
  • Contoh
    =IF(A1>70, "Lulus", "Tidak Lulus")

    Jika nilai di sel A1 lebih dari 70, maka hasilnya "Lulus", jika tidak maka "Tidak Lulus"

  • Contoh Lainnya
    =IF(AND(A1>=0, A1<=100), "Valid", "Invalid")

    Memeriksa apakah nilai di sel A1 berada dalam rentang 0-100

  • Contoh dengan Text
    =IF(UPPER(A1)="YES", "Disetujui", "Ditolak")

    Memeriksa jika teks di sel A1 adalah "YES" (tidak case sensitive)

  • Contoh dengan Tanggal
    =IF(TODAY()>A1, "Sudah Lewat", "Masih Berlaku")

    Memeriksa apakah tanggal di sel A1 sudah lewat dari hari ini

  • Contoh dengan Perhitungan
    =IF(A1*2>100, "Besar", "Kecil")

    Melakukan perhitungan sebelum membuat keputusan

  • Contoh dengan ISBLANK
    =IF(ISBLANK(A1), "Data Kosong", "Data Terisi")

    Memeriksa apakah sel A1 kosong atau sudah terisi

  • Contoh dengan ISNUMBER
    =IF(ISNUMBER(A1), A1*2, "Bukan Angka")

    Jika sel A1 berisi angka, kalikan dengan 2, jika tidak, tampilkan pesan error

IF Bersarang (Nested IF)

  • Sintaks:
    =IF(test1, value1, IF(test2, value2, value3))
  • Contoh:
    =IF(A1>=80, "A", IF(A1>=70, "B", IF(A1>=60, "C", "D")))

    Menentukan grade nilai berdasarkan rentang nilai tertentu

  • Contoh Status Karyawan:
    =IF(A1="Tetap", IF(B1>=5, "Senior", "Junior"), IF(A1="Kontrak", "Kontrak", "Magang"))

    Menentukan status karyawan berdasarkan jenis (Tetap/Kontrak) dan masa kerja

  • Contoh Bonus Penjualan:
    =IF(A1>1000000, IF(B1>12, "Bonus 15%", "Bonus 10%"), IF(A1>500000, "Bonus 5%", "Tidak Ada Bonus"))

    Menghitung bonus berdasarkan nilai penjualan dan masa kerja dalam bulan

  • Contoh Status Pembayaran:
    =IF(A1="Lunas", "Selesai", IF(B1>TODAY(), "Dalam Tenor", IF(C1>0, "Cicilan Terlambat", "Belum Bayar")))

    Menentukan status pembayaran berdasarkan kondisi pelunasan, tanggal jatuh tempo, dan cicilan

  • Contoh Kategori Produk:
    =IF(A1="Elektronik", IF(B1>5000000, "Premium", "Standar"), IF(A1="Pakaian", IF(C1="Import", "Premium", "Lokal"), "Lainnya"))

    Mengkategorikan produk berdasarkan jenis, harga, dan asal produk

  • Contoh Status Kelulusan:
    =IF(AND(A1>=75, B1>=80), "Lulus Dengan Distinction", IF(AND(A1>=60, B1>=65), "Lulus", IF(OR(A1<60, B1<65), "Tidak Lulus", "Data Tidak Valid")))

    Menentukan status kelulusan berdasarkan nilai teori (A1) dan praktik (B1)

  • Contoh Status Stok:
    =IF(A1=0, "Stok Habis", IF(AND(A1>0, A1<=B1), "Perlu Restock", IF(A1>B1, "Stok Aman", "Data Tidak Valid")))

    Menentukan status stok barang berdasarkan jumlah stok tersedia (A1) dan batas minimum stok (B1)

  • Contoh Nilai Akhir:
    =IF(AND(A1>=0,A1<=100), IF(B1>=0.7, A1*B1, IF(B1>=0.5, A1*B1*0.8, A1*B1*0.6)), "Nilai Tidak Valid")

    Menghitung nilai akhir berdasarkan nilai ujian (A1) dan tingkat kehadiran (B1)

  • Contoh Prioritas Pengiriman:
    =IF(A1="Express", IF(B1<=5, "Kirim Hari Ini", "Kirim Besok"), IF(A1="Regular", IF(C1>1000000, "Prioritas", "Antrian Normal"), "Tipe Tidak Valid"))

    Menentukan prioritas pengiriman berdasarkan tipe layanan (A1), jumlah antrian (B1), dan nilai pesanan (C1)

Rumus SUM

SUM Dasar

  • Sintaks:
    =SUM(number1, [number2], ...)
  • Contoh:
    =SUM(A1:A10)

    Menjumlahkan semua nilai dalam rentang sel A1 sampai A10

SUM dengan Kondisi

  • SUMIF:
    =SUMIF(range, criteria, [sum_range])
  • Contoh:
    =SUMIF(B1:B10, ">=1000", C1:C10)

    Menjumlahkan nilai di range C1:C10 jika nilai di B1:B10 lebih besar atau sama dengan 1000

  • Contoh Penjualan Bulanan:
    =SUM(B2:B13)

    Menjumlahkan total penjualan selama 12 bulan (dari sel B2 sampai B13)

  • Contoh Multiple Range:
    =SUM(A1:A10,C1:C10,E1:E10)

    Menjumlahkan nilai dari tiga kolom berbeda sekaligus

  • Contoh dengan Nilai Tetap:
    =SUM(A1:A5,100,200)

    Menjumlahkan range sel dengan nilai tetap tambahan

  • Contoh Matriks:
    =SUM(A1:C3)

    Menjumlahkan semua nilai dalam matriks 3x3

  • Contoh Non-Adjacent:
    =SUM(B2,B4,B6,B8)

    Menjumlahkan nilai dari sel-sel yang tidak bersebelahan

  • Contoh dengan Fungsi Bersarang:
    =SUM(MAX(A1:A5), MIN(B1:B5))

    Menjumlahkan nilai maksimum dari range A1:A5 dengan nilai minimum dari range B1:B5

  • Contoh dengan Referensi 3D:
    =SUM(Sheet1:Sheet12!A1)

    Menjumlahkan nilai sel A1 dari Sheet1 hingga Sheet12

  • Contoh dengan Operator Perbandingan:
    =SUM((A1:A10>0)*(A1:A10))

    Menjumlahkan hanya nilai positif dalam range A1:A10

  • Contoh dengan Named Range:
    =SUM(Penjualan_Q1, Penjualan_Q2)

    Menjumlahkan nilai dari dua named range yang telah didefinisikan

  • Contoh dengan Offset:
    =SUM(OFFSET(A1,0,0,5,2))

    Menjumlahkan nilai dalam area dinamis 5 baris x 2 kolom dimulai dari sel A1

Rumus VLOOKUP

VLOOKUP Dasar

  • Sintaks:
    =VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])
  • Contoh Pencarian Harga Produk:
    =VLOOKUP(A2, Products!A2:C100, 2, FALSE)

    Mencari harga produk berdasarkan kode produk di kolom A2

  • Contoh dengan Range Lookup:
    =VLOOKUP(B5, Grades!A2:B10, 2, TRUE)

    Mencari grade nilai berdasarkan rentang nilai

  • Contoh dengan Multiple Sheets:
    =VLOOKUP(A1, Sheet2!$A$1:$D$100, 4, FALSE)

    Mencari data dari sheet lain dengan referensi absolut

  • Contoh dengan IFERROR:
    =IFERROR(VLOOKUP(A2, Database!A:D, 3, FALSE), "Data Tidak Ditemukan")

    Menangani error jika data tidak ditemukan

  • Contoh dengan Named Range:
    =VLOOKUP(CustomerID, CustomerDatabase, 3, FALSE)

    Menggunakan named range untuk memudahkan referensi

  • Contoh Nested VLOOKUP:
    =VLOOKUP(VLOOKUP(A1, Table1!A:B, 2, FALSE), Table2!A:C, 3, FALSE)

    Menggunakan hasil VLOOKUP sebagai input untuk VLOOKUP lainnya

HLOOKUP Dasar

  • Sintaks:
    =HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])
  • Contoh Data Kuartalan:
    =HLOOKUP("Q2", A1:E3, 2, FALSE)

    Mencari data penjualan untuk kuartal kedua

  • Contoh dengan Multiple Rows:
    =HLOOKUP(B1, Data!A1:F4, 3, FALSE)

    Mencari data dari baris ketiga berdasarkan header

  • Contoh dengan Text Match:
    =HLOOKUP("*" & A1 & "*", Database!A1:E5, 4, FALSE)

    Mencari data dengan partial match menggunakan wildcard

  • Contoh dengan Named Range:
    =HLOOKUP(ProductCode, ProductMatrix, 2, FALSE)

    Menggunakan named range untuk tabel pencarian

  • Contoh dengan IFERROR:
    =IFERROR(HLOOKUP(C1, Table!A1:D4, 3, FALSE), "No Data")

    Menangani error dengan pesan kustom

  • Contoh dengan Multiple HLOOKUP:
    =HLOOKUP(A1, Sheet1!A1:E5, 2, FALSE) + HLOOKUP(A1, Sheet2!A1:E5, 2, FALSE)

    Menjumlahkan hasil dari dua HLOOKUP berbeda

Shortcut Microsoft Word

Shortcut Dasar

  • Ctrl + C: Menyalin teks
  • Ctrl + V: Menempel teks
  • Ctrl + X: Memotong teks
  • Ctrl + Z: Membatalkan perubahan
  • Ctrl + Y: Mengulangi perubahan

Shortcut Formatting

  • Ctrl + B: Teks tebal
  • Ctrl + I: Teks miring
  • Ctrl + U: Teks garis bawah
  • Ctrl + E: Teks tengah
  • Ctrl + J: Teks rata kiri-kanan

Navigasi Teks

  • Ctrl + →: Pindah satu kata ke kanan
  • Ctrl + ←: Pindah satu kata ke kiri
  • Home: Ke awal baris
  • End: Ke akhir baris
  • Ctrl + Home: Ke awal dokumen

Manajemen Dokumen

  • Ctrl + N: Dokumen baru
  • Ctrl + O: Buka dokumen
  • Ctrl + S: Simpan dokumen
  • Ctrl + P: Cetak dokumen
  • Ctrl + F: Cari teks

Format Lanjutan

  • Ctrl + D: Font dialog box
  • Ctrl + Shift + >: Perbesar ukuran font
  • Ctrl + Shift + <: Perkecil ukuran font
  • Ctrl + L: Rata kiri
  • Ctrl + R: Rata kanan

Fungsi Khusus

  • Ctrl + Enter: Page break
  • Ctrl + Shift + 8: Tampilkan tanda format
  • Alt + Ctrl + N: Navigation pane
  • Alt + Ctrl + I: Preview dokumen
  • Ctrl + Shift + C: Salin format

Shortcut Microsoft Excel

Navigasi

  • Ctrl + ↑: Ke sel teratas kolom
  • Ctrl + ↓: Ke sel terbawah kolom
  • Ctrl + Home: Ke sel A1
  • F5: Membuka dialog Go To
  • Ctrl + Page Up/Down: Pindah worksheet
  • Ctrl + Home: Ke sel A1
  • Ctrl + End: Ke sel terakhir yang berisi data

Formula & Data

  • F2: Edit sel aktif
  • F4: Mengulang perintah terakhir
  • Alt + =: AutoSum
  • F4: Mengunci referensi sel ($)
  • Ctrl + `: Tampilkan rumus/hasil
  • Alt + H + O + I: AutoFit lebar kolom
  • Alt + H + O + A: AutoFit tinggi baris
  • Ctrl + T: Buat tabel
  • Alt + F11: Buka VBA Editor
  • Ctrl + Shift + L: Toggle filter
  • Alt + F1: Buat grafik
  • Alt + W + F + F: Bekukan panel

Formatting & Tampilan

  • Ctrl + 1: Format sel
  • Ctrl + Shift + ~: Format angka umum
  • Ctrl + Shift + $: Format mata uang
  • Ctrl + Shift + %: Format persen

Manipulasi Data

  • Ctrl + Shift + L: Filter data
  • Ctrl + -: Hapus sel/baris/kolom
  • Ctrl + +: Tambah sel/baris/kolom
  • Ctrl + Space: Pilih seluruh kolom
  • Shift + Space: Pilih seluruh baris

Fungsi Lanjutan

  • Ctrl + Shift + &: Tambah border
  • Alt + F1: Buat grafik
  • F11: Buat grafik di sheet baru
  • Alt + F8: Tampilkan macro
  • Ctrl + Shift + #: Format tanggal

Pivot Table

  • Alt + N + V: Buat PivotTable
  • Alt + F5: Refresh PivotTable
  • Alt + JT: Buat PivotChart
  • Alt + D + P: Tampilkan Field List

Analisis Data

  • Alt + D + S: Sort Data
  • Alt + D + F: Filter Data
  • Alt + D + G: Group Data
  • Alt + D + U: Ungroup Data

Shortcut File Explorer

Navigasi

  • Windows + E: Buka File Explorer
  • Alt + D: Fokus ke address bar
  • Alt + ←: Kembali ke folder sebelumnya
  • Alt + →: Maju ke folder berikutnya
  • Alt + ↑: Ke folder induk
  • Backspace: Kembali ke folder sebelumnya

Operasi File

  • Ctrl + C: Salin file/folder
  • Ctrl + X: Potong file/folder
  • Ctrl + V: Tempel file/folder
  • F2: Rename item terpilih
  • Delete: Hapus ke Recycle Bin
  • Shift + Delete: Hapus permanen

Seleksi & Tampilan

  • Ctrl + A: Pilih semua item
  • Ctrl + Mouse Click: Pilih multiple item
  • Shift + Mouse Click: Pilih item berurutan
  • F11: Layar penuh
  • Alt + P: Panel preview
  • Ctrl + Shift + N: Buat folder baru

Fitur Lanjutan

  • F4: Buka address bar dropdown
  • Alt + Enter: Lihat properti item
  • Windows + P: Pilih tampilan proyeksi
  • F5: Refresh jendela
  • Alt + Tab: Beralih antar jendela
  • Windows + L: Kunci komputer
